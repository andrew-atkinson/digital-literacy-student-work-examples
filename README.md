# SP24 Student Work — ARFD 106 Digital Literacy

Static gallery page for student work examples from the Digital Literacy Foundation Course, Spring 2024.

## Setup

1. Place all student asset files in the `assets/` folder (they must match these exact filenames):

```
assets/
├── bellaiyana_172077_17232870_Creativeproject.jpeg
├── fischerthomas_LATE_151324_17343103_fischer_creativeproject1Large.jpeg
├── khateebamal_LATE_157391_17682593_Midsemester Project-Khateeb.png
├── vidalnicky_LATE_149518_17825174_cambridgevidalrevised2.png
├── bellaiyana_LATE_172077_17812264_redux_proj_final.mp4
└── patelpriya_152744_17796261_attentioncomp.mp4
```

2. Open `index.html` in a browser — or deploy the whole folder to any static host (GitHub Pages, Netlify, etc.).

## Notes

- **Work 4 (Data Sale)** has no file extension in the original filename. Once you know the format, add the correct extension to both the filename in `assets/` and update the `src` attribute in `index.html` (search for `cambridgevidalrevised2`).
- Images open in a fullscreen lightbox on click.
- Videos use the native browser `<video>` player with controls.
- The filter bar at the top lets viewers browse by project.

## Structure

```
arfd106-student-work/
├── index.html      ← the whole site
└── assets/         ← drop all student files here
```
